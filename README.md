# StudentTaxCalculator
## Tax calculator for students

A calculator build to help students understand where thier income is going, and how much money they will receive after state taxes is taken out. As a student worker at a university or college, it is essential to understand where income is going and what you are being taxed on. 

This project started as a personal tool to help me calculator my earnings as a student worker in Illinois, but it can be used by any student in a state with a stable income tax percentage. 

Those students who are in states with no income tax or graduated-rate income tax, will unfortunately not be able to use this calculator.
